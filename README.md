## About The Project

[![Product Name Screen Shot][product-screenshot]](https://github.com/JaninaMattes/EdgeAI-Object-Detection)

This project is an experimental platform designed to explore the trade-offs between model size and inference performance on mobile hardware. By connecting a FastAPI "Optimization Engine" to a native iOS "Telemetry Probe," we can visualize how pruning and quantization affect real-world metrics like FPS, thermal state, and battery drain.

## Getting Started

### Prerequisites

* **Backend**: Install `uv` for 10x faster Python project management.
  ```sh
  curl -LsSf [https://astral.sh/uv/install.sh](https://astral.sh/uv/install.sh) | sh
  ````

* **Frontend**: Xcode 15+ and an iPhone with an A13 chip or newer (for Neural Engine testing).

### Installation

1. Clone the repo
   ```
    git clone [https://github.com/JaninaMattes/EdgeAI-Object-Detection.git](https://github.com/JaninaMattes/EdgeAI-Object-Detection.git)
   ```
2. Backend setup
   ```
    cd backend
    uv sync
    uv run uvicorn app.main:app --reload
   ```
3. Frontend Setup
   * Open ```ios-app/EdgeAI.xcodeproj``` in Xcode.
   * Update the BackendConfig.swift with your machine's local IP address.
   * Build and run on your physical iPhone.
